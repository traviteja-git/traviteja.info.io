## Overview

Engineered a batch ingestion pipeline for Hux, a Customer Data Platform (CDP) built for enterprise clients at Deloitte. The platform consolidates customer data from multiple sources into a unified profile for analytics and activation.

## Key Contributions

- Designed and built batch ingestion DAGs using Apache Airflow (Cloud Composer) to collect data from 10+ source systems
- Implemented data quality validation gates before loading into BigQuery
- Optimised pipeline scheduling to reduce processing windows and cloud compute costs
- Collaborated across engineering, data science, and client teams during a multi-year engagement

## Impact

- **25%** reduction in cloud infrastructure costs
- **20%** increase in pipeline scalability
- Reduced time-to-insight for business analysts from days to hours
